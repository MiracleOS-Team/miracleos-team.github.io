Why MiracleOS?

Works Out of the Box

The OS is made to work Out of the Box, without any necessary user intervetion for it to work

User First Model

In MiracleOS, a good experience for the user is always the first thing we lookout for, independent if they are an experienced user or not

Gaming

Play Steam games with Valve's Proton compability layer for Linux

Big selection of apps

MiracleOS provides a big selection of apps for the user, such as Steam and Firefox

Rock Solid Stability

MiracleOS provides rock solid stability, free from any system instability

Easy Rollbacks

The BTRFS filesystem allows you to rollback a system to a certain point in time if anything bad happens
